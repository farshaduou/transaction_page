# transaction_page
web-based solution to accept and settle decentralized transactions

## description of project:
this project is to be deployed separately and called whenever other projects need to conduct a transaction with the user. they will call this project, receive a way of payment via decentralized methods, like BTC on-chain or lightning network or stable-coins.

## types of transactions accepted:
### Bitcoin on-chain transactions
### Bitcoin lightning network transactions
### Stablecoins transactions
#### USDT
